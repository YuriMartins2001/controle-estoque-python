# 📦 Controle de Estoque em Python

Sistema de controle de estoque desenvolvido em Python com foco em aprendizado de lógica de programação, manipulação de arquivos JSON e organização de código.

## 🚀 Funcionalidades

- ✅ Adicionar produtos
- ✅ Remover produtos
- ✅ Consultar produtos
- ✅ Visualizar estoque completo
- ✅ Gerar código automático para cada produto
- ✅ Persistência dos dados em arquivo JSON
- ✅ Tratamento de erros para entradas inválidas

## 🛠️ Tecnologias utilizadas

- Python 3
- JSON
- Git
- GitHub

## ▶️ Como executar

```bash
python EstoqueYuri.py
```

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar conceitos fundamentais de programação em Python, como:

- Funções
- Dicionários
- Laços de repetição
- Tratamento de exceções (`try` / `except`)
- Manipulação de arquivos JSON
- Organização de código
- Versionamento com Git e GitHub
